    目前成果

低功耗广域网

具有低功耗、海量分布的优点，针对可以忍受一定延时的应用，具有很好的适应性。目前包括LoRa、NB-IoT在内的技术，正具有良好的发展势头。

人工智能

对海量数据进行分析，科学计算各项指数，达到智能化的精确决策与控制。其核心是搭建智能化的分析模型，使用真实数据进行训练。

异构组网

使用多种物联网协议同时工作，将智能设备接入互联网，同时实现海量节点与低功耗共存，传输速率与数据量相匹配。在控制成本的情况下实现最优应用方案。

Android工具

Android系统既可运行于手机，又可运行于专门为物联网服务的嵌入式设备，作为网关或管理员设备都是非常合适的选择，App的开发是物联网非常重要的一环。

4G实时应用

目前可以直接接入互联网的协议中，4G的传输速率是最快的。因此对于需要实时监控、数据量较大的终端，使用4G进行传输是很合适的。

Windows软件

Windows是世界上最普及的系统，在学生、研发者和企业管理者群体中的熟练程度都很高，使用Windows系统的管理软件，可以使参与者更易接受，帮助系统更好的运行。

欢迎走进我们的研究

物联网与人工智能，即是我们的专业，也是我们的爱好。在学习中创新，在创新中获取技术的升华。成为程序猿，就要不断研发。
